:mod:`timer` --- Timers
=======================

.. module:: timer
   :synopsis: Timers.

----------------------------------------------

.. class:: timer.Timer(timeout, event, mask=-1, flags=0)

   Initialize given timer object. On timeout given `mask` is written
   to given `event` channel.


   .. method:: start()
    
      Start the timer.
    

   .. method:: stop(value)
    
      Stop the timer.
