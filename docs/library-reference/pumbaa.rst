:mod:`pumbaa` --- Pumbaa
========================

.. module:: pumbaa
   :synopsis: Pumbaa.

The pumbaa module is a thin layer on top of `Simba` to make most
`Simba` functionality available in `Python`. It is not intended to be
compatible with `MicroPython:s` hardware modules as of today.
